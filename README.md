# Global-warming
Это простой проект, который помогает работать с текстами о глобальном потеплении.

Программа позволяет:
- переводить текст на разные языки (русский, английский, немецкий, французский),
- считать количество символов в тексте.
- два мини теста
В проекте используется готовый текст о глобальном потеплении, который можно перевести или проанализировать.
